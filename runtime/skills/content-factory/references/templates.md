# 内容运营模板

## 内容简报

```markdown
# 内容简报 — [品牌/项目]

## 概述
[一句话描述]

## 目标平台
- 主平台：[平台]
- 辅助平台：[平台]

## 受众画像
- 人口统计：[年龄、性别、城市线级]
- 心理特征：[兴趣、痛点]
- 内容偏好：[他们喜欢消费什么内容]

## 成功指标
- [平台]：[KPI目标]

## 品牌调性
[语气、风格、人设]

## 发布节奏
- [平台]：[频率]

## 竞品参考
- [竞品]：[他们做得好的地方]
```

## 数据报告

```markdown
## 数据报告 — [品牌] — 第[N]周

### [平台]
发布：[N]篇 | 核心指标：[数值]
最佳："[标题]" — [指标] — 原因：[分析]
最差："[标题]" — [指标] — 改进：[措施]

### 跨平台内容复用
[平台]最佳内容"[标题]" -> 下周改编投放到[目标平台]

### 下周计划
- 加大[平台]投入
- 测试[新内容形式]
- 复盘[失败案例]
```
