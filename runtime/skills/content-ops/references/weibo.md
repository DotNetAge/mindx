# 微博写作规范 (Weibo Writing Standard)

> 本文件是 content-ops 技能的**平台参考标准**，不是独立技能。当需要为微博生产内容时，由 SKILL.md 的工作流按需加载本文件。

## 平台本质

微博 = **中国舆论广场**

速度第一，传播第二，深度第三。微博的核心价值在于 **时效性** 和 **传播速度**，不在于单条内容有多深。

## 核心机制

### 时效性 = 生命线

```
热点的半衰期：热门话题可能只有几小时
普通内容的有效期：12-24小时
精华内容的长尾：2-3天
```

**在微博，"慢一步"等于"没发"。**

### 1400字的物理限制

微博的字数限制迫使 **极致精炼**。每一条字都必须有价值。

### 转发是核心指标

```
一条微博的真实价值 = 原始互动 × 转发链条长度
```

- 100个赞 < 10个转发（转发带来新的曝光，赞不会）
- 转发链条越长，内容触达的人群越大
- 设计内容时要考虑"这条会不会被人转发"

### 话题标签（#XXX#）是流量入口

```
带话题 = 加入某个流量池
不带话题 = 只在自己的粉丝圈里传播
```

- 蹭热度要 **自然**，硬蹭适得其反（会被认为是营销号）
- 每条微博 **1-3个话题标签** 足够（多了显得杂乱）

## 一、内容类型与模板

### Type A: 热点点评型（最常见、最核心）

微博的灵魂就是 **"对正在发生的事有态度"**。

```markdown
{事件概述（1句话，假设读者已经知道背景）}

{你的观点/角度（1-2句话，要有态度！）}
→ 不要中立！微博要有立场才有转发
→ 可以犀利、可以幽默、可以温暖、可以愤怒
→ 但绝对不能无聊（"各有各的道理" = 没人说）

{延伸思考/补充信息（可选，1-2句）}

#{热点话题} #{相关话题} #{个人话题标签}
```

**关键：** 微博的转发动力 = **情绪共鸣** 或 **"他说出了我想说的话"**。

### Type B: 干货分享型

```markdown
【{主题}】{核心结论/数字}

1. {要点一}（一行内说完）
2. {要点二}
3. {要点三}
...

→ 详细版请看：{公众号/小红书/知乎链接}
（微博=引流入口，深度内容放其他平台）

#{话题标签}
```

**定位：** 微博是 **"预告片"**，其他平台是 **"正片"**。

### Type C: 日常/人设型

```markdown
{一个真实的、有画面感的日常片段/感悟}
→ 要有"人味儿"，不要太官方
→ 可以是小吐槽、小确幸、工作瞬间、生活观察
→ 最好有画面感（让读者能在脑中想象场景）

（配图/视频增强真实感）
```

**目的：** 建立 **人格化的品牌形象**，而不只是一个冷冰冰的账号。

### Type D: 互动/提问型

```markdown
{一个问题或选择}
A. {选项A}
B. {选项B}
C. {选项C}

你们选哪个？评论区见！

#{话题}
```

**技巧：** 选择题比开放式问题更容易获得评论（降低了参与门槛）。

### Type E: 金句/格言型

```markdown
"{一句有力量的、值得转发的话}"

—— {署名/出处/或原创标注}
```

**最佳场景：** 这种内容最容易获得"转发到朋友圈"（因为短、有逼格、能代表转发者的品味/态度）。

## 二、微博写作铁律

### 1. 第一句就抓住注意力

微博信息流极快，用户平均在每条微博上停留 **不到2秒**。第一句必须完成钩子的使命。

### 2. 有态度 > 正确但无聊

- "这件事双方都有道理" → 没人转发
- "这件事A明显有问题，原因是..." → 有人转发（即使不同意）
- 微博不是写论文的地方，**有锋芒的内容才有人看**

### 3. 极致精炼

- 删掉所有"我认为""我觉得""我个人觉得"
- 删掉所有过渡句和连接词
- 一句话说清一件事
- 如果说不清，说明你想说的太多了（拆成两条）

### 4. 配图/视频是必须的

- 纯文字微博在信息流中 **几乎不可见**
- 图片：高清、有信息量、或者有情绪感染力
- 视频：15-60秒，竖屏优先
- 图文搭配时，文字不要太长（盖过图片）

### 5. 话题标签用法

```
正确：#{精确话题} #{相关话题}
错误：#{超级泛的话题} #{完全不相关的热门话题} #{十个标签}
```

## 三、发布时机矩阵

| 时段 | 适合内容 | 原因 |
|------|---------|------|
| **7:00-9:00** | **早安/新闻/正能量** | **通勤第一波，刷手机高峰** |
| **11:30-13:00** | **午餐/轻松/话题讨论** | **午休时间，讨论意愿高** |
| **17:30-19:00** | **下班/吐槽/娱乐** | **晚高峰，情绪释放** |
| **20:00-23:00** | **深度/热点/互动** | **黄金时段，活跃度最高** |
| **0:00-2:00** | **深夜情感/夜猫子** | **年轻用户活跃，情感内容效果好** |
| **热点随时发** | **热点点评** | **时效性 > 最佳时段** |

### 热点发布的特殊规则

```
发现热点 → 15分钟内评估 → 值得跟 → 30分钟内发出
                                              ↓
                                    超过2小时 = 基本错过
```

**热点速度公式：**
- 第一波（0-2小时）：信息传播阶段 → 快速跟进有红利
- 第二波（2-6小时）：观点发酵阶段 → 有独特角度仍有价值
- 第三波（6小时+）：总结/反思阶段 → 深度复盘有长尾价值

## 四、微博人格建设

虽然微博是碎片化的，但长期来看，成功的微博账号都有一个 **鲜明的人格**。

### 人格三要素

| 要素 | 含义 | 如何体现 |
|------|------|---------|
| **立场** | 你站在哪边？ | 热点事件中的一致性态度 |
| **风格** | 你怎么说话？ | 语言风格（犀利/温暖/幽默/理性） |
| **关注点** | 你在乎什么？ | 持续关注的领域/话题方向 |

### 人格一致性检验

```
随便挑你最近10条微博，问：
1. 如果遮住头像和名字，能认出是你写的吗？
2. 这些微博看起来像是同一个人写的吗？
3. 别人能用一句话描述你的微博风格吗？

如果任何一个答案是"否"，说明人格还不够鲜明。
```

## 五、跨平台角色

在微博生态中，这个账号的角色应该是：

```
其他平台的"广播站"和"引路人"
        ↓
微博 = 把其他平台的精华提炼成碎片
        ↓
引导感兴趣的用户回到主阵地（公众号/小红书/知乎）
```

**但要注意：** 微博不能只是"链接搬运工"。每条微博都必须有 **独立价值** —— 即使不点链接，读者也从这条微博中获得了一些东西（信息/情绪/启发）。

## 六、质量检查卡

> 每条微博发布前，必须逐项通过此检查卡。

```
微博发布前自查清单
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【内容质量】
□ 有明确的立场/态度/信息增量？（不能是正确的废话）
□ 第一句就能抓住注意力？
□ 极致精炼（删掉了所有冗余）？
□ 字数在有效范围内（1400字内，最佳200-500字）？

【形式】
□ 有配图或视频？（纯文字曝光极低）
□ 图片/视频清晰且有信息量？
□ 带了合适的话题标签（1-3个）？

【时机】
□ 发布时机匹配内容类型和目标用户？
□ 如果是热点，时效性还在吗？（别追已经凉了的热点）

【人格】
□ 符合账号的一贯风格/立场？
□ 有"人味儿"（不像官方通告/营销文案）？

【合规】
□ 没有违规/敏感内容风险？
□ 如涉及争议话题，表述有分寸？
□ 没有误导性信息？

【传播潜力】
□ 有人想要转发这条的理由吗？（情绪/价值/认同/吐槽）
□ 如果被转发到朋友圈，新读者能看懂吗？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
传播预估：⭐⭐⭐☆☆ (3/5)
是否通过：✅ YES / ❌ 需修改（原因：__________）
```
