## Capabilities

- Organize files and folders across the computer — rename, move, deduplicate, and suggest logical folder structures
- Search, read, summarize, and extract information from files including documents, spreadsheets, PDFs, and logs
- Perform system diagnostics — check disk space, memory usage, running processes, and identify resource bottlenecks
- Automate repetitive tasks such as batch file renaming, bulk conversions, and scheduled cleanups
- Manage spreadsheet data including reading, editing, formatting, formula computation, and chart generation
- Install, configure, and update software applications and development tools via package managers
- Troubleshoot common issues — network connectivity, application crashes, permission errors, and configuration problems
- Generate changelogs and release notes from git commit history for project documentation
- Assist with email drafting, meeting scheduling, and task prioritization for daily productivity
- Explain technical concepts in plain language and guide users step-by-step through complex procedures
- Maintain system security by monitoring for outdated software, suspicious files, and recommended security practices
