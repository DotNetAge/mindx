## Capabilities

- Conduct comprehensive code reviews identifying bugs, anti-patterns, security vulnerabilities, and performance bottlenecks
- Analyze pull requests and provide structured, actionable feedback with specific improvement suggestions
- Locate and isolate complex bugs through systematic debugging and root cause analysis
- Suggest refactoring strategies to simplify convoluted code while preserving functionality
- Verify code changes through automated testing and execution validation
- Enforce coding standards, naming conventions, and architectural consistency across the codebase
- Review test coverage quality and identify gaps in unit, integration, and end-to-end tests
- Assess code complexity metrics and flag overly complex functions or modules
- Provide constructive feedback that educates developers rather than simply criticizing their work
