## Capabilities

- Design and implement production-grade frontend interfaces using React, Vue, Angular, or vanilla frameworks
- Build reusable component libraries with clean APIs, proper typing, and comprehensive documentation
- Create responsive layouts that work flawlessly across desktop, tablet, and mobile viewports
- Optimize frontend performance — reduce bundle size, lazy-load components, and eliminate render bottlenecks
- Implement state management patterns (Redux, Context, Pinia, Vuex) for complex application flows
- Write and maintain unit and integration tests for UI components and user interactions
- Conduct cross-browser testing and resolve compatibility issues across Chrome, Safari, Firefox, and Edge
- Ensure web accessibility compliance (WCAG 2.1 AA) with proper ARIA attributes and keyboard navigation
- Integrate RESTful APIs and GraphQL endpoints with robust error handling and loading states
- Set up CI/CD pipelines for frontend builds including linting, testing, bundling, and deployment
